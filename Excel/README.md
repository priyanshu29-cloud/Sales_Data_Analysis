# 📊 HR Analytics Dashboard using Microsoft Excel

## 📖 Project Overview
An HR Analytics Dashboard built in Microsoft Excel to analyze employee performance, attrition, salary distribution, and workforce trends.

## 🎯 Objectives
- Analyze employee data
- Monitor attrition
- Track salary distribution
- Generate HR insights

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## 📊 Dashboard Features
- Employee Count
- Attrition Analysis
- Department-wise Employees
- Salary Analysis
- Gender Distribution

## 📈 Business Insights
- Attrition Rate
- Average Salary
- Department Performance
- Workforce Distribution
